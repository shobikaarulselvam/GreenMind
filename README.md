# 🌿 GreenMind — AI-Powered Smart Plant Care App

GreenMind is a full-featured, single-file web app that helps home gardeners 
in India track, manage, and grow their plants smarter using AI.

## Features
- 🔐 User authentication — Sign up, login, Google auth, persistent sessions
- 🌱 My Garden — Add & track plants with smart watering schedules
- 💧 Watering Tracker — Due alerts, climate-adjusted water amounts
- 📅 Growth Timeline — Visual plant growth history
- 🌤️ AI Climate Report — Daily gardening report powered by Groq AI (llama-3.3-70b)
- 🤖 AI Advisor — Chat with an expert Indian horticulturalist AI
- ✨ Recommendations — Soil & climate-based plant suggestions
- 👥 Community — Post tips, questions, harvests; AI answers on every post
- 🏆 Rewards & XP — Gamified streaks and achievement badges
- 👤 Profile & Settings — Edit avatar, manage notifications, export data

## Tech Stack
- Pure HTML, CSS, JavaScript (no frameworks, single file)
- Groq AI API — llama-3.3-70b-versatile for all AI features
- LocalStorage for persistent data
- Location & climate-aware (Tamil Nadu / India focused)

## Setup
1. Download `GreenMind.html`
2. Open in any browser
3. Start growing 🌱
